# tibudget API

## ? Currently in development

This project is still under development and is not ready for production use.

---

## ? Installation (via JitPack)

### 1. Add the JitPack repository

    <repositories>
        <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>
    </repositories>

### 2. Add the dependency

    <dependency>
        <groupId>com.github.tibudget</groupId>
        <artifactId>tibudget-api</artifactId>
        <version>0.51</version>
    </dependency>

---

## ?? Versioning

Versions are based on Git tags.

Make sure the version you use matches an existing tag:

    0.49
    1.0.0
